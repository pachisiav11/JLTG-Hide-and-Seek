Install the APK (Android)

  1. 1Tap Download the Android app above (on your phone).
  2. 2Open the downloaded JLTG.apk. Android will ask to allow installing from this source — tap Settings → enable Allow from this source, then go back and tap Install.
  3. 3Open JLTG H&S from your app drawer.
  4. 4On first launch it asks for a Google Maps API key — paste yours. It is stored only on your device and never leaves it.

The app is a thin wrapper around the live web app, so it always runs the latest version and works offline once loaded. It may briefly show a small address bar at the top — that is normal for this kind of installed web app.

Simpler: install straight from Chrome (no APK)

You can also install it as an app without the APK — this version runs fully full-screen and auto-updates:

  1. 1Open the web app in Chrome on your phone.
  2. 2Menu ⋮ → Add to Home screen / Install app.
  3. 3Launch it from your home screen like any app.

Getting a Maps API key

  • Create a project at Google Cloud Console and enable Maps JavaScript, Directions, Distance Matrix, and Places APIs.
  • Create an API key under Credentials and paste it into the app on first launch.
  • Personal use stays within Google's free monthly tiers. If you ever host this publicly, restrict the key by HTTP referrer.

What you can do

  • Zones — add named regions (Singapore, Switzerland), draw, or import; combine into a play area.
  • Questions — Radar, Thermometer, Matching, Tentacles, Measuring; shade the map down to the hider.
  • Hider zone, Directions, Transit, Measure, and permanent local game history with export/import.
